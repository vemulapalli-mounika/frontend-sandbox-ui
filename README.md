# Frontend Sandbox UI

## 🚀 Project Overview
This is a UI design recreated from a Figma design using HTML and CSS.

## 🛠️ Technologies Used
- HTML5 (Semantic structure)
- CSS3 (Flexbox & Grid)
- Basic JavaScript

## 🎯 Features
- Responsive layout
- Modern UI design
- Reusable card components
- Highlighted feature section

## 🧠 Concepts Used
- Flexbox for layout alignment
- CSS Grid for card structure
- Semantic HTML for clean structure
